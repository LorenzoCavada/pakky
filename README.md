# Pakky

Pakky, the best solution to securely order your products.

## Install Dependencies

```bash
npm i
```

## Available Scripts

### Development

```bash
npm run start
```

### Production

```bash
npm run build
```

### About

Made with [Alpine](https://github.com/alpinejs/alpine) and [Tailwind](https://tailwindcss.com/) to let HTML shine.
